## Usage

### Compile OSO to IR

```julia
using .OsoCompiler

source = """
@impact(ase=5.0) {
}
"""

ir = OsoCompiler.compile(source)
println(ir)  # JSON IR
```

### Execute IR

```julia
using .OsoVM

results = OsoVM.execute_ir(ir)
println(results)  # [{"opcode": "IMPACT", "ase_minted": 23.265}]
```

### Run Tests

```bash
julia test/oso_vm_test.jl
```
